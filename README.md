# eCommerce API
- Node/Express REST API to provide typical functionality found in an eCommerce website
- Users can create accounts, view products, add products to a cart, and place/view orders

## Objectives
- Building a functioning eCommerce REST API using Express, Node.js and PostgreSQL
- Allowing users to register and log in via the API, and to place an order
- Allowing CRUD operations on products, user accounts, user carts, and orders
- Using Git version control
- Using command line 
- Developing locally on your computer
- Documenting the API using Swagger
- Deploying the API live to Heroku
